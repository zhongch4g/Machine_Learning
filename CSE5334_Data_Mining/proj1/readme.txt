This code apply python3 and need import numpy module.


How to run this code:
	In terminal enter "python3 LinearRegression.py"

Change learning rate or EPOCH of stochastic gradient descent?
In kfold() function to change learning rate or EPOCH.

Change initial theta?
In SGD() function to change theta.

Change K of the k-fold cross validation?
In run() function to change the K of the k-fold cross validation.
